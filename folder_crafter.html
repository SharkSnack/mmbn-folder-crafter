How to Use

1. Select game.
2. Click on chips
　to add to folder.
3. Right-click on a
　chip to remove it.
4. Export folder to
　share with others.
5. Import folder to
　edit others' chips.

Notes

Hover over code to
see chip locations.

Hover over name to
see chip description
and memory size.
